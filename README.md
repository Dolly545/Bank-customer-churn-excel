# Bank-customer-churn-excel
**Project Objective**
To analyze customer churn behavior in a retail banking dataset and identify key factors influencing customer retention using Excel, Power Query, and Pivot Tables.

**Tools & Skills Used**
Microsoft Excel
Power Query (Data Cleaning & Transformation)
Pivot Tables & Pivot Charts
Data Analysis & Business Insights
KPI Calculation

**Dataset Description**
The dataset contains customer demographic and financial information including:
Age, Gender, Geography
Account Balance
Number of Products
Credit Card Status
Activity Status
Estimated Salary
Churn Indicator

**Additional derived columns were created:**
Age Group
Churn Status (Member of Bank / Exit from Bank)
Activity Status (Active / Inactive)
Credit Card Status

**Data Cleaning & Preparation (Power Query)**

Removed unnecessary columns
Checked and handled null values
Standardized categorical values
Created calculated columns for segmentation (Age Group, Activity Status, Credit Card Status)
Ensured consistent data types

**Business Questions Answered**

What is the churn rate by number of products?
How does churn vary across age groups?
Do active members churn less than inactive members?
Does having a credit card impact churn?
Which geography shows higher churn risk?

**Relationship between balance and churn behavior**

** Key Insights**

Customers with higher number of products show different churn patterns.
Inactive members have higher churn risk compared to active members.
Customers in certain age groups show higher exit rates.
Geographical differences impact churn behavior.
Credit card ownership shows correlation with retention.

**Outcome**

This project demonstrates:
End-to-end Excel analytics workflow
Data cleaning using Power Query
Business-driven pivot analysis
Insight generation for decision-making
